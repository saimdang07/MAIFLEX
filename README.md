# MAIFLEX
Maiflex adalah aplikasi kredit pinjaman tunai berbasis mobile, pada folder ini terdapat beberapa design seperti:
1. Architecture System
2. Screen Flow
3. User Story
4. ERD
5. Flowchart
6. Spec API
berkaitan dengan scope: Registrasi, verifikasi akun, pembuatan PIN, Login dan Pencairan, termasuk melihat jumlah tagihan, terimakasih.
